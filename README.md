# Compilador-Gramatical-N-o-Sem-ntico-Aplicado-Comunica-o-de-Baixo-N-vel-com-LLMs-PECTOR-
O Pector é um projeto de engenharia de linguagem e métodos formais projetado para atuar como um intermediário determinístico de escopo fechado entre instruções humanas em recursos naturais e Modelos de Linguagem (LLMs) ou sistemas de automação de infraestrutura.
